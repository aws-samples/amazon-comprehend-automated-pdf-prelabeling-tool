ACToolkit Demo

Introduction

An intentionally simple demo to demonstrate how to embed ACToolkit into your own webpage.

How to Use

TL;DR: it is as easy as it can get.

First add the following css and javascript link into <head> section of your html first: 

    
    <link href="https://fonts.googleapis.com/css?family=Roboto+Condensed:700|Roboto:300,400,700|Maven+Pro:400,500,700|Oswald:400,500,700"  rel="stylesheet">
    <link href="https://drive-render.corp.amazon.com/view/tiayuche@/actoolkit/0.7.1/actoolkit.css"
          rel="stylesheet">
    <script src="https://drive-render.corp.amazon.com/view/tiayuche@/actoolkit/0.7.1/actoolkit.js"></script>

The body of your html looks like this: 

    
    <body>
    <div id="tagging"> </div>
    <body>

Last, add the following code to an additional <script> tag. 

    
    <script>
    var d = {"raw_text": "This is an example document"}
    var store = new ACToolkit.TaggingStateStore(d, "discovered_coins", tagSet = ["BITCOIN", "ETH", "$", "GOLD"]);
    var node = document.getElementById("tagging");
    ACToolkit.createTaggingAnnotator(store, node);
    </script>

To programmatically assess current annotations of the document, just do: 

    
    var annotations = store.annotations;
